#Portfolio & Prompt Engineering Showcase

Hi — welcome to my portfolio repository. This repo is organized to showcase:
- AI Prompt Engineering work (prompts, variations, evaluation & results)
- Short projects and demos that demonstrate product thinking and execution
- Readable case-style writeups that recruiters and hiring managers can skim

Repository layout
- README.md — this overview (you are here)
- PORTFOLIO.md — curated portfolio entries with templates you can reuse
- prompts/ — curated prompt collections, experiments, and evaluation notes
- examples/ — runnable examples, small scripts and demo code
- docs/ — static site content for GitHub Pages (optional)
- CONTRIBUTING.md — how to add new portfolio entries or prompts

How to use this repo
1. Browse PORTFOLIO.md for the high-level stories.
2. Go to prompts/ for the raw prompts, versions, and evaluation results.
3. Use docs/ to power a GitHub Pages site (Settings → Pages → Deploy from branch → /docs).
